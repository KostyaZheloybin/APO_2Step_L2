# APO_2Step_2L

Лабораторная работа №2 «Каркас мультиагентной системы» по дисциплине «Архитектура программного обеспечения»

Для расширения работы калькулятора (также как и в первой лабораторной 
работе) был добавлен актор – OrepationAgent, который хранит 
математическую операцию (произведение, сумма чисел, среднее значение, 
очистка списка, вывод списка) и отправляет сообщение актору калькулятор 
для выполнения математической операции. Логику агента математических 
операций реализовали в файле operation_agent.py. Также был доработан 
агент-калькулятор.
